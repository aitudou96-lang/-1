## 通用标注要求

- 更适合账号：AI先锋 / AI先锋者 / 双账号均可

使用标准：

- 主表只保留强相关和中相关内容，弱相关 AI 资讯不进入主表

## 日报输出结构

每日输出建议按以下顺序：

- 一、AI先锋 今日变现内容包：至少 1 条流量内容、1 条信任内容、1 条转化内容
- 二、AI先锋者 今日IP内容包：至少 1 条流量型认知内容、1 条信任型方法论内容、1 条长期人设内容
- 三、今天最建议拍的4条
- 四、今日信息源摘要
- 五、今日不建议做的内容
- 六、今日读取失败的信息源
- 七、明天建议观察什么
- 八、合规边界

“今天最建议拍的4条”必须固定包含：

- AI先锋 2 条：1 条流量/信任优先，1 条转化优先
- AI先锋者 2 条：1 条流量型认知优先，1 条长期信任/人设优先

每条必须包含：

- 标题
- 前3秒
- 账号
- 内容目的
- 拍摄形式
- 结尾方式
- 是否提产品
- 为什么今天优先拍
- 拍摄难度
- 预计价值

结构约束：

- 已进入 AI先锋 今日变现内容包的流量、信任、转化内容，不再在日报后半部分重复用“今日适合做流量/信任/转化的内容”重新展开
- 日报后半部分只保留信息源摘要、风险/不建议内容、读取失败、明日观察和合规边界
- “两个都适合”只作为底层判断保留；日报推荐区必须给出更明确的主账号归属

## 内容建议输出结构

每条内容建议必须包含：

- 标题方向
- 开头钩子
- 核心观点
- 内容结构
- 适合账号
- 内容目标
- 驱动类型
- 推荐形态
- 是否承接 AI口播智能体
- 承接方式：私信 / 直播 / 教程演示 / 案例拆解 / 观点铺垫 / 暂不承接
- 产品连接句
- 不适合承接原因
- 下一步动作
- 一句话拍摄建议

AI先锋 输出补充要求：

- 每日必须给出流量、信任、转化各 1 条候选
- 可以承接 799 元 AI口播智能体，但必须写清承接方式和结尾 CTA
- 内容目标围绕精准流量、信任建立、私信/直播/教程/案例转化

AI先锋者 输出补充要求：

- 每日必须给出流量型认知、信任型方法论、长期人设各 1 条候选
- 默认不强提 799 产品
- 内容目标围绕 AI认知、AI变现观察、趋势判断、误区拆解、个人实战记录、编导和运营方法论
- AI先锋者 与 AI先锋 同等重要，不作为变现号附属
