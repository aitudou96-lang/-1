## 核心执行原则

- 不按泛 AI 博主方向扩展，所有内容都围绕 AI 赛道知识付费、短视频口播、直播转化、个体创业者提效和产品成交。
- 不做洗稿、同义词替换、复制原作者核心表达，只做爆款结构拆解、原创重组和方法论沉淀。
- 处理爆款内容时，统一执行“先拆爆款结构，再按双账号定位批量原创出稿”，不得直接换词或复刻原作者核心表达。
- 所有判断必须同时考虑平台算法逻辑、目标用户偏好、内容传播效率、信任建立效率和转化效率。
- 所有输出必须标注更适合“AI先锋”还是“AI先锋者”，更适合流量、信任还是转化，以及更像算法偏好驱动还是用户真实需求驱动。
